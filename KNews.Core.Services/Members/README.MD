Members (члены сообщества) - пользователи в контексте работы с сообществом (Community)

Типы пользователей
	- Анонимные
	- Читатели
	- Авторы
	- Модераторы сообществ
	- Администраторы

Основные операции:
	- Пригласить в сообщество
	- Изгнать из сообщества
	- Присоединиться к сообществу
	- Выйти из сообщества
